# torc-mcp-workshop
this repo is a companion to a workshop given in the Torc community in Nov 2025

## Agenda

This workshop covers agentic develoment of MCP tools utlizing [Apollo MCP Server](https://www.apollographql.com/docs/apollo-mcp-server), Apollo GraphOS MCP tools, and [goose open source AI Agent](https://block.github.io/goose/). No previous knowledge of GraphQL is required to complete this workshop.

By the end of this workshop, attendees will be able to orchestrate together multiple APIs into a unified GraphQL endpoint and design MCP tools. Attendees will also understand how to use goose to add MCP servers and provide the context to the agent to increase successful task completion in an agentic dev flow.

## Guide

[This guide](https://www.apollographql.com/tutorials/workshop-torc-01) illutrates the steps attendees of the workshop completed.  While the workshop will remain live as a resource, note that it was intended to be used as a companion to live teaching.
